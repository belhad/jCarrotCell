Setup

Download the jCarrotCell bundle or clone the repo from Github. You will need to include some version of jQuery (version 1.7+) before jCarrotCell.js and also the styles from carrotcell.css.


  • Include CSS

    Include the carrotcell.css , or carrotcell.scss to set the default styles of carrotcell. You can manually edit the css file to change the styles, or over ride them by pass in additional CSS classes through the settings object. 

    <link rel="stylesheet" type="text/css" href="carrotcell.css">

  • Create Markup

    Markup (using list)

    <ol class="my-carrotcell">
    <li>item 1</li>
    <li>item 2</li>
    <li>item 3</li>
</ol>

    OR Markup (using divs)

    <div class="my-carrotcell">
    <div>item 1</div>
    <div>item 2</div>
    <div>item 3</div>
</div>

    You can use whatever tags you like as long as they follow the basic structure of a parent element including some children elements.


  • Include Javascript

    jCarrotCell uses Velocity.js for animations and fall back on jQuery animate. If you want to use something other than the default tweens with animate and NOT velocity, please include jQuery Easing , otherwise, jCarrotCell will use the default tween "swing".

    Javascript includes (the minimal): 

    <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js" type="text/javascript" charset="utf-8"></script>
<script src="jcarrotcell.min.js" charset="utf-8"></script>

    OR Javascript includes with Velocity (recommended):

    <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js" type="text/javascript" charset="utf-8"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/velocity/1.2.2/velocity.min.js" type="text/javascript" charset="utf-8"></script>
<script src="jcarrotcell.min.js" charset="utf-8"></script>

    OR Javascript includes with Jquery Easing:

    <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js" type="text/javascript" charset="utf-8"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery-easing/1.3/jquery.easing.min.js" type="text/javascript" charset="utf-8"></script>
<script src="jcarrotcell.min.js" charset="utf-8"></script>

  • Create CarrotCell

    Create a carrotcell by calling the function .carrotCell on a jQuery selector. You can check the console for any errors. 

    $(".my-carrotcell").carrotCell();

Settings

Change the settings of the carrotcell by passing in a settings object in the carrotcell creation.

Using Settings

$(".my-carrotcell").carrotCell({
    show: 3,
    scroll: 3
});

Settings Values

  • show (integer, default 1)

    How many items to show on screen

  • scroll (integer, default 1)

    How many items to scroll

  • duration (integer millisecs, default 500)

    Length of scroll animation duration

  • sideways (boolean, default true)

    True indicates sideways or horizontal scrolling, false indicates vertical scrolling

  • easing (string, default "easeOutExpo" or "swing")

    The tween animation for scrolling, "easeOutExpo" is used if Velocity or Easing is included, "swing" if only jQuery Animate.

  • infinite (boolean, default false)

    Indicates this will scroll infinitely in both directions. If using auto: true this will be triggered to true.

  • usePrevNext (boolean, default true)

    Generate previous and next buttons

  • prevClass

    Class name for previous button

  • prevIconClass

    Class name for prevous icon, inside previous button

  • prevText (string, "next item set")

    Accessible text for previous, inside previous button

  • nextClass

    Class name for next button

  • nextIconClass

    Class name for next icon, inside next button

  • nextText (string, default "previous item set")

    Accessible text for next, inside next button

  • auto (boolean, default false)

    Indicate this slide show will auto play

  • autoDuration (integer millisecs, default 10000)

    How long to pause if auto scrolling on each item

  • pauseOnHover (boolean, default true)

    Hovering over the carrotcell will pause the auto animation

  • usePausePlay (boolean, default true)

    Generate pause and play buttons if using auto

  • pauseClass (string)

    Class name for pause button

  • pauseIconClass (string)

    Class name for pause icon, inside pause button.

  • pauseText (string, default "pause auto scroll")

    Accessible text for pause button.

  • playClass (string)

    Class name for play button

  • playIconClass (string)

    Class name for play icon, inside play button.

  • playText (string, default "resume auto scroll")

    Accessible text for play button.

  • controlOnHover (boolean, default false)

    Hide the previous, next, play, pause controls unless carrotcell is being hovered over. This is disabled on touch devices.

  • useDots (boolean, default false)

    Generate the dots used for navigation. This is calculated per scroll event.

  • dotsClass (string)

    Pass in a class name for the dots container. Please use "someclass someother-class" and not ".someclass"

  • dotClass (string)

    Class name for the individual dots, this is inside dotsClass.

  • dotButtonClass (string)

    Class name for the dot button, this is inside dotClass.

  • dotIconClass (string)

    Class name for the dot icon, this is inside dotButtonClass.

  • dotText (string, default "item set " + n)

    This is the access text for dots, ths is inside dotButtonClass. It will read "item set 0", "item set 1" and so on.

  • onClass (string, default "carrotcell--on")

    This is the class used when the dot is highlighted when the item it is associated with is in view.

  • dotsOnHover (boolean, default false)

    Hide the dots navigation unless carrotcell is being hovered over. This is disabled on touch devices.

  • key (boolean, default false)

    Use keys to navigate controls

  • keyBack (integer, default 37 left arrow or 38 up arrow)

    The keycode used to navigate previous

  • keyForward (integer, default 39 right arrow or 40 down arrow)

    The keycode used to navigate next

  • keyToggle (integer, default 80 letter p)

    The keycode used to toggle pause or play

  • breakpoints (array of objects)

    Pass in additional parameters on pixel breakpoints for window width

    $(".my-carrotcell").carrotCell({
    breakpoints : [
        { pixels: 320, settings: { scroll: 1, show: 1 },
        { pixels: 900, settings: { scroll: 2, show: 2 }},
        { pixels: 1010, settings: { scroll: 3, show: 3 }}
    ]
});

API

Getting the API

There are several ways to get the carrotCell API. When carrotCell is invoked, it returns the API object. 

var myAPI = $("#my-carrotcell").carrotCell();
myAPI.moveToItem(3); // use the api to scroll to item 3

If you are are calling carrotCell on more than one object, the return variable will be array. This is a javascript native array, not a jQuery object array. 

var myAPIs = $(".my-carrotcell").carrotCell();
// [API-1, API-2, ... ]

The API is stored on the data attribute "carrotcell__api" on an element.

var myAPI = $(".my-carrotcell").data("carrotcell__api");
myAPI.moveToItem(3);

// or you can use it directly

$(".my-carrotcell").data("carrotcell__api").movetoItem(3); // scroll to item 3
