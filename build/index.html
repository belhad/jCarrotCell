A flexible, responsive, accessible jQuery carousel plugin.

Features

  • Horizontal, vertical, infinite, auto, scroll by n scrolling.
  • All controls and slides are accessible via keyboard and touch/swipe.
  • Insert and remove items, update carousel config, move to item via the API.
  • Responsive to window resize, set width breakpoints for different configs.
  • Customize with CSS styles, or pass in CSS classnames to over ride defaults.

Setup

Download the jCarrotCell bundle or clone the repo from Github.

Include CSS:

<link rel="stylesheet" type="text/css" href="carrotcell.css">

Write Markup:

    
    
  1. item 1
    2. 
    
  2. item 2
    3. 
    
  3. item 3
    4.

Include Javascripts: 

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js" type="text/javascript" charset="utf-8"></script>
<script src="jcarrotcell.min.js" charset="utf-8"></script>

Call jCarrotCell:

$("#my-carousel").carrotCell();

Checkout the documentation for more information on settings and API. Or look at the demos.
