Circular Auto Advance Scrolling by 1 Slide Per Page

By setting auto to true, the carousel automatically becomes circular. The cloned elements are set with a yellow border to be more easily identified. The page size of this is set to be 1 item per page and the auto advance pauses when you hover over the carousel.


$('#kitten5').carrotCell({ 
	auto: true, 
	pauseOnHover: true, 
	delay: 1000, 
	step: 1 
});

Arrow Key Advance Scrolling and External Page Navigation

Using the left and right arrow keys to navigate should be reserved for pages with 1 carrotCell object with key set to true since it is a document wide event. Vertical carrotCells use the up and down arrow keys. The external .navi is still enclosed in .carrotCell but can bet set to whatever element you wish through the configuration.

Note that if you click the navi buttons quickly, the carousel will not recognize the event until the scrolling animation is done.


$('#kitten6').carrotCell({ 
	key: true,  
	navi: true 
});

Using Different HTML Containers

Some of the default HTML containers can be changed to the user's preference. Please consult the documentation for more information.

Note that parent-child hierarchy inside carrotCell must be observed other wise the default javascript selectors will fail. For instance: .carrotCellView must be inside the .carrotCell, and another element must wrap the individual slides in place of the UL. The slides must also be direct children of the UL replacement


<div id="kitten8Nav">
	<div id="goBack" class="prev">prev</div>
	<div class="navi">
		<div>1</div>
		<div>2</div>
		<div>3</div>
	</div>
	<div id="goForward" class="next">next</div>
</div>

<div id="kitten8" class="carrotCell">
	<div class="carrotCellView">
		<div>
			<div><div>1</div><img src="images/cat01.png"></div>
			<div><div>2</div><img src="images/cat02.png"></div>
			...

		</div>
	</div>
</div>

The controls for the carousel can exist in a different container than carrotCell but you must specifically pass it containsControl boolean and the controlScope. If no controlScope is specified, the search is conducted on the root BODY element.


$('#kitten8').carrotCell({
	sliderSelect : "div",
	sliderChildSelect : "div",
	containsControl : false,
	navi : true,
	controlScope : "#kitten8Nav",
	prevSelect : "#goBack",
	nextSelect : "#goForward"
});
