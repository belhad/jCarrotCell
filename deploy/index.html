A flexible, responsive, customizable carousel plugin for jQuery.

Features

  • Use your own markup, event names, classes, selectors etc.
  • Scroll vertically or horizontally, infinitely or finitely.
  • Auto play, pause, stop.
  • Auto generate "dots" or configure your own.
  • Scroll multiple items, or just one.
  • Add and remove items via the API.
  • Auto fill max width of browser, auto scale the height too, or don't.
  • Arrow key navigation.
  • Use touch events when possible, or don't.
  • Supports orientation change in maxwidth mode.

Setup

Download the jCarrotCell bundle or clone the repo from Github.

You will need to include some version of jQuery (version 1.7+) before jCarrotCell.js and also the styles from carrotcell.css.

The markup should look something like this, see the Documentation on how to use whatever markup you want.

<div class="carrotCell"> <div class="carrotCellView"> <ol> <li>item 1</li> <li>item 2</li> <li>item 3</li> <li>item 4</li> <li>item 5</li> <li>item 6</li> <li>item 7</li> <li>item 8</li> <li>item 9</li> </ol> </div> <div class="prev">previous</div> <div class="next">next</div> </div>

Invoke the carrotcell with the following:

$('.carrotCell').carrotCell();
