What Is It?

jCarrotCell is a jQuery carousel plugin that does all the popular carousel things.

What Makes It Different?

  • Flexible markup - you don't need to use any DIVs if you don't want to.
  • Pass in custom selectors to find the controls, you don't need to put them in a specific place - but you do have to define how to find them.
  • Have the code auto-generate the paging navigation for you, or customize it if you want to.
  • Insert / Remove items, empty the whole carousel, reload the carousel with new items with AJAX or whatever and have it actually work.
  • Nested scrollers.
  • Touch swipe support on applicable devices.
  • Kittens, puppies.

What Can It Do?

  • Vertical / horizontal scrolling.
  • Circular / infinite scrolling.
  • Scrolling speed, scrolling delay control.
  • Timed auto advance.
  • Play, pause, and stop control in auto advance mode.
  • Pause auto advance on hover.
  • Stop on click of a navigation element.
  • Navigate using custom key events.
  • Clickable paging navigation or "navi".
  • Exposed API methods.
  • Custom Events.
  • Custom class names.
  • Custom selectors to find controls anywhere on the page.

Where Is It?

You can get it on github https://github.com/jyuya/jCarrotCell.

2. Include the carrotcell.css.


<link type="text/css" href="css/carrotcell.css" rel="stylesheet">

3. Give the carrotCell some styling.


.carrotCell { 
	width: 400px; 
}		
.carrotCell .disabled  { 
	color: #ccc; 
}
.prev, .next {
	display: block;
}
.prev { 
	float: left; 
	margin-right: 320px;
}

4. Include jQuery (1.8.3).

5.Include jCarrotCell.js file after jQuery.


<script src="jCarrotCell.js" type="text/javascript" charset="utf-8"></script>

6. Invoke carrotCell() on the .carrotCell on document ready.


$(function() {
    $('.carrotCell').carrotCell();
});

Optional. You can also enable swipe support for the carousel on touch devices (iPhone, iPad) simply by including the jquery-touchwipe.js by Andreas Waltl. This enables you to swipe on the carousel to advance forward or back.


<script src="jquery-touchwipe.js" type="text/javascript" charset="utf-8"></script>
