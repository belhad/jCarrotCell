What Is It?

jCarrotCell is yet another jQuery carousel plugin, because there isn't enough of them out there.

Features include:

  • Vertical / horizontal scrolling
  • Circular / infinite scrolling with speed control
  • Auto advance on a custom timer
  • Play, pause, and stop in auto advance mode
  • Append to / remove from list at specific spot
  • External navigation support
  • Exposed API methods
  • jQuery selectors as custom controls
  • kittens

Where Is It?

You can get it on github https://github.com/jyuya/jCarrotCell.

How Do I Use It? (TL,DR Version)

1. Set up your carousel content in HTML. Here is an example of a basic layout.

<div id="myCarousel" class="carrotCell">
     <div class="carrotCellView">
          <ol>
               <li>...</li>
               <li>...</li>
               ...
          </ol>
     </div>
     <div class="prev">prev</div>
     <div class="next">next</div>
     ...
</div>

2. Include the carrotcell.css or import the styles into your own style sheet.

<link type="text/css" href="css/carrotcell.css" rel="stylesheet">

3. Include jQuery, jCarrotCell was developed on jQuery v1.5.2 but 1.6.2 seems to work as well... for now.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.6.2/jquery.min.js" type="text/javascript" charset="utf-8"></script>

4. Now include the jCarrotCell.js file

<script src="jCarrotCell.js" type="text/javascript" type="text/javascript" charset="utf-8"></script>

5. Finally, call the jCarrotCell function on the element that is of class ".carrotCell"

<script type="text/javascript">
     $(document).ready(function(){
           $('.carrotCell').carrotCell();
     });
</script>

Of Note. There is sometimes a bug in Chrome where document ready would fire before the elements are rendered with the correct height, here is an alternate version of 5 that fixes the problem.

$(document).ready(function(){
     window.someNameSpace = window.someNameSpace || {};
     someNameSpace.setup = function() {
          $('.carrotCell').carrotCell();
     }
     var is_chrome = navigator.userAgent.toLowerCase().indexOf('chrome') > -1;
     if (is_chrome) {
          $(window).load(function(){ someNameSpace.setup(); });
     } else {
          someNameSpace.setup();
     }
});
